## Required Access
The script requires:
- GitHub repository write access for `indigo-services/var1-home-landing`
- GitHub project scopes for `gh` (`read:project` and project write capability)

## Run
```powershell
pwsh ./ops/github/bootstrap-release-v0.1.0.ps1
```

If project scopes are still unavailable but repo write access exists, run:

```powershell
pwsh ./ops/github/bootstrap-release-v0.1.0.ps1 -SkipProject
```

That will still create the milestone, labels, and backlog issues.
